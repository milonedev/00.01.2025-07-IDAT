Hackaton 03

1.- Hacer un algoritmo en JavaScript que lea un número por el teclado y determinar si tiene tres dígitos. 


let numero = Number(inputEjercicio1.value);

if (isNaN(numero)) {
  console.log("Debe ingresar un número válido.");
} else {
  let valorAbsoluto = Math.abs(numero);
  if (valorAbsoluto >= 100 && valorAbsoluto <= 999) {
    console.log("El número tiene tres dígitos.");
  } else {
    console.log("El número NO tiene tres dígitos.");
  }
}

2.- Hacer un algoritmo en JavaScript que lea un número entero por el teclado y determinar si es negativo. 


let numero = Number(inputEjercicio1.value);

if (isNaN(numero)) {
  console.log("Debe ingresar un número válido.");
} else {
  if (numero > 0) {
    console.log("El numero No es negetivo.");
  } else {
    console.log("El numero es negetivo.");
  }
}

3.- Hacer un algoritmo en JavaScript que lea un número y determinar si termina en 4. 


let numero = Number(inputEjercicio1.value);

if (isNaN(numero)) {
  console.log("Debe ingresar un número válido.");
} else {
  let valorAbsoluto = Math.abs(numero);
  if (valorAbsoluto % 10 === 4) {
    console.log("El numero termina en 4");
  } else {
    console.log("El numero no termina en 4");
  }
}
